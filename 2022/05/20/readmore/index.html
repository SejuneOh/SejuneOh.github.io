Hexo 더 읽어보기 버튼 사용하기

넥스트 테마를 사용하면서 일단… 많은 기능들을 찾아보고 사용하고 있지만 역시 hexo는 어렵다..

포스트 자체를 제목과 날짜만 표시하고싶은데 해당 방법을 찾지 못해서 일단 너무 긴 글은 더 읽어 보기 버튼을 만들려고한다.

일단 config 파일에서 더 보기 버튼이 활성화 되어있는지 확인하자

테마 _config.yml 옵션 확인

나는 Next 테마를 사용하고있는데 각자의 테마의 Document를 읽어 보고 설정하는 것을 권장한다.

../node_module/hexo-theme-[테마이름]/_config.yml 파일에서 아래와 같은 항목을 찾자

read_more_btn: true 해당 항목이 true로 되어 있다면 활성화 되어있는 것 !!!

next 테마에서는 일정 버전 이후에는 기능이 작동하지 않는 것으로 나와있다…. 좌절…

하지만 hexo-excerpt 플러그인을 사용하면 정상 작동한다고 한다 .

Plugin 설치

실행해보니 정상적으로 작동한다. 차근차근 하나씩 블로그를 꾸며봐야겠다.

개발을 공부하면서 hexo도 같이 공부하니깐 블로그도 개발하는 느낌이들어서 조금 재밌다.
